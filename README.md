# NovaStore Database Project

Bu proje, NovaStore isimli bir e-ticaret sistemi için SQL Server kullanılarak oluşturulmuş veri tabanı tasarımını içermektedir.

## İçerik
- Veri tabanı oluşturma
- Tablo tasarımı ve ilişkiler
- Veri ekleme (DML)
- SQL sorguları ve analiz
- View oluşturma
- Veri tabanı yedekleme

## Dosya Yapısı
- `sql/` → SQL scriptleri
- `docs/` → Proje raporu

## Final SQL Script
Tüm işlemleri içeren birleşik SQL dosyası:
`sql/NovaStore_Final.sql`
